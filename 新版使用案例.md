# VirtualKey模块更新
> 新增`scancode_down_up`和`keybd_event`方法, 均为驱动级模拟.


## 导入模块
```
from bdtime import tt       # 新版由 bd_time 改为 bdtime
from VirtualKey import keybd_event, scancode_down_up, scancodes, down_up
```

## scancode_down_up
```
def f():
    ch = 'a'
    ret = scancode_down_up(ch)
    return ret


tt.sleep(0.5)
print(f())
# scancodes.get('a') --> 获取 按键a 的扫描码
```

## keybd_event
```
# keybd_event
def f():
    ch = 's'
    ret = keybd_event(ch)
    return ret


tt.sleep(0.5)
print(f())
```

## down_up
```
def f():
    ch = 'd'
    ret = down_up(ch)
    return ret


tt.sleep(0.5)
print(f())
```
